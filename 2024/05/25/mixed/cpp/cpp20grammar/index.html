# std::string

这个可以让我们更方便地处理 string 字符串格式化的问题，学过 Python 的话对这个会更熟悉。

首先需要引入头文件
#include <format>

# 最简单的情况：

std::string formatted_string = std::format("My name is {} and I am {} years old.", name, age);

# 对小数格式化：

float x = 1.151, float y = 3.15
std::string formatted_string = std::format("coordination: ({:.1f}, {:.1f})", x, y);

输出：
coordination: (1.2, 3.1)
我们可以注意到当小数后面省去的超过一半时是会进位的，0.15 会省略尾数变为 0.1，0.151 就是 0.2 了。

# 带索引的占位符

std::string formatted_string = std::format("Name: {0}, Age: {1}, Name: {0}", name, age);
输出：
Name: name, Age: age, Name: name

# 带索引同时对小数格式化

后面的变量可以是表达式的形式，支持加减乘除。
std::string formatted_string = std::format("Product: {0}, Price: ${1:.2f}, Quantity: {2}, Total: ${3:.2f}", product, price, quantity, price * quantity);
