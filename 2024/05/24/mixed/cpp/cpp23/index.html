# MinGW 下载

可以在下载地址里下载最新版本：

下载 x86_64-13.2.0-release-posix-seh-ucrt-rt_v11-rev1.7z 即可。

然后将其解压，如下图示，将 mingw64 拷贝到自己存放文件的盘，不建议存放在 C 盘，笔者将其放在 D 盘的 Tools 文件夹下，如下图示。

# 配置环境变量

在电脑下方的搜索框里输入环境就可以出来。

然后点击高级 - 环境变量 - 双击（系统变量里的）Path - 将 mingw64 的 bin 目录输入进去，然后一路确定关闭即可。
![](envi1.webp 配置环境)
调 cmd 出来输入 gcc -v 查看是否有版本号输出。
再输入 gcc -v -E -x c++ - 后续会用到下面的路径，可以提前复制。

# 配置 VScode

进入 VScode 以后按下 ctrl+shift+p，输入 c/c++，选择 Edit Configuration (UI)

将刚才复制的路径粘贴到 Include Path 中
